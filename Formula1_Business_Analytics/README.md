Formula 1 Racing Analytics Dashboard

# Project Overview
This project is an interactive Tableau dashboard built using Formula 1 historical racing data. It provides insights into driver performance, constructor success, race trends over time, and circuit distribution across different countries.

# Dashboard Features
- Top 10 Drivers by Total Points
- Top Constructors by Total Points
- Number of Races by Year
- Circuits by Country
- KPI Cards:
  - Total Drivers
  - Total Constructors
  - Total Races
  - Total Circuits

# Tools Used
- Tableau Public
- Excel/CSV
- GitHub

# Dataset
Formula 1 historical racing dataset (1950–2024)
